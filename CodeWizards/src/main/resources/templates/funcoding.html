Funcoding

- A fun filled coding sessions

LEARNING PHASE: includes two levels

Level
It was intended to kickstart the Coding journey of students. Acquiring basic knowledge on a programming language being it's main motto it will assure to enhance the problem solving skills of the students by 50 suggested problems by the tutors.
Level
After completing Level 1, students will be proficient with the syntax of a language.Level 2 Focuses more on logic. We will explore multiple possible solutions and analyse their complexities by that students can approach to any problems in an optimized way. (includes 100 suggested programs). Later students will build simple stuff like console applications, stimulating games and etc.

Level-1

Syllabus

  1. Getting Started with python
  2. Flow of Control
  3. Introduction to Non-Primitive Data types.
  4. Functions

Pre-Requisites
  1. Basic math concepts such as arithmetic operators, primes, divisors, multiples
  2. Should be able to spend 2 hours for the session and practice.
  3. A will to learn and a chill for mind:)
Features
  1. Daily Sessions
  2. Daily Curated Practice Problems
  3. 24*7 on chat Doubt Support
  4. Daily Doubt Resolution Sessions
  5. Networking with like-minded peers

Level-2

Syllabus

  1. Getting started with Algorithms and flowcharts.
  2. Introduction to problem solving
